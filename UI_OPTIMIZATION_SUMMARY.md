### 2️⃣ Daily Summary 页面重设计

**文件**: `src/pages/daily-summary.tsx`

#### 核心优化：

##### 🌊 蓝绿渐变背景
```css
bg-gradient-to-br from-teal-50 via-cyan-50 to-emerald-50
```
- 柔和的蓝绿色调，营造宁静的书写氛围
- 渐变过渡自然，不会过于刺眼

##### 🫧 浮动泡泡装饰
添加三个动态浮动泡泡：
- **泡泡1**: `bg-teal-200/30` - 顶部左侧，8秒浮动周期
- **泡泡2**: `bg-cyan-200/20` - 顶部右侧，10秒浮动周期（延迟2秒）
- **泡泡3**: `bg-emerald-200/25` - 底部中间，12秒浮动周期（延迟4秒）

动画效果：
```css
@keyframes float {
  0%, 100% { transform: translateY(0) translateX(0); }
  50% { transform: translateY(-20px) translateX(10px); }
}
```
